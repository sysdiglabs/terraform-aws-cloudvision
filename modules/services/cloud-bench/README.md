# Cloud Bench deploy in AWS Module

Deploys

- the required IAM Role and IAM Policies to allow Sysdig to run AWS Benchmarks on your behalf
- An `aws_foundations_bench-1.3.0` benchmak task schedule on  `0 6 * * *`
